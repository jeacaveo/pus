PUS
===

Frontend site for Prismata related information (strategy game).

Development environment
-----------------------

0. Virtual environment (look at virtualenvwrapper):

   Create:

    mkvirtualenv pus

   Activate:

    workon pus

1. Install dependencies:

    pip install -r requirements-dev.txt

2. Run tests:

    ./runtests.sh

3. Run linters:

    ./runlinters.sh
